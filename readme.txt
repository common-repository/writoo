== Description ==
**Français**

Writoo est un plugin qui vous rédigera des articles en automatique. Vous renseignez vos mots clés et nos rédacteurs se feront un plaisir de réaliser votre demande.
Vous aurez ainsi des articles uniques sur Internet. Tous les articles créés vous appartiennent.


*  Les articles sont rédigés par des rédacteurs
*  Vous recevez vos articles sont aucunes interventions
*  Paiement sécurisé

== Installation ==

1. Uploader le fichier writoo.zip et decompressez le dans le dossier /wp-content/plugins/writoo/
2. Activez the plugin dans la rubrique extension de WordPress
3. Paramétrez vos mos clés dans la rubrique Writoo (Réglages)
